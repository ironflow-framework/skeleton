# IronFlow Skeleton

Ce dépôt est le squelette officiel pour démarrer un projet avec IronFlow.

## Installation

```bash
composer create-project ironflow/skeleton nom-du-projet
```

## Structure

- `app/` : Modules métiers
- `bootstrap/app.php` : Initialisation du Kernel
- `config/` : Configurations
- `public/index.php` : Entrypoint web
- `routes/web.php` : Routes de base
- `storage/` : Logs et cache
